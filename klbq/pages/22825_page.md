[首页](https://wiki.biligame.com/klbq/%E9%A6%96%E9%A1%B5 "首页")  **信用分**

[短](https://wiki.biligame.com/klbq/?curid=22825)[刷](./22825_page.md)[编](./22825_page.md)[历](./22825_page.md)

为维护稳定游戏环境，在S9赛季【隙间重构】中，游戏内将上线【信用分】系统。系统将基于引航者的游戏行为，设置不同信用等级体系，形成有效约束力。

致力于保护正常游戏玩家的同时，惩罚消极/恶意游戏玩家，从源头减少不良游戏行为，为广大引航者的良好游戏体验提供有利保障。

在引航者的宿舍主界面，点击左上角个人头像进入个人信息界面，即可看见【信用分】分区。

## 信用评级

### 信誉评级

共划分6级，初始等级未定级，但享有4级同等权限。各级分别对应不同的名称及颜色，且分别对应的行为限制也有所不同。信誉评级未定级玩家参与10场任意模式比赛后即可定级（不含自定义房间）。

| 等级 | 可游玩模式 | 其它权限 |
| --- | --- | --- |
| 1级：恶劣 | 【自定义】、【团队乱斗】 |  |
| 2级：不良 | 【自定义】、【挑战赛所有模式】 |  |
| 3级：合格 | 【自定义】、【挑战赛所有模式】、【排位赛】 | 聊天、语音 |
| 4级：良好 | 【自定义】、【挑战赛所有模式】、【排位赛】 | 聊天、语音 |
| 5级：优秀 | 【自定义】、【挑战赛所有模式】、【排位赛】 | 聊天、语音、额外奖励时装 |
| 6级：完美 | 【自定义】、【挑战赛所有模式】、【排位赛】 | 聊天、语音、额外奖励时装、额外奖励勋章 |
| 未评级 | 【自定义】、【挑战赛所有模式】、【排位赛】 | 聊天、语音 |

### 语音文明评级

共划分4类，不同类型会有相应处罚。

| 类型 | 权限 |
| --- | --- |
| 表现良好 | 文字和语音频道不受限制 |
| 已有违规 | 文字和语音频道暂时不受限制 |
| 发言受限 | 因多次不文明发言遭到禁言 |
| 发言受限 | 因信用评级低，文字语言功能受限 |

### 语音文明惩罚梯度

当玩家累计语音文明违规次数过多时，惩罚时间将按梯度递增。

| 惩罚等级 | 处罚限制时间 |
| --- | --- |
| 警告1 | 24小时 |
| 警告2 | 48小时 |
| 警告3 | 72小时 |
| 警告4 | 168小时 |

## 违规内容

| 类型 | 说明 | 处罚 |
| --- | --- | --- |
| 消极比赛 | 指在对局中故意不攻击对手、故意不操控自己的超弦体、故意使自己的超弦体战败等不以获得游戏胜利为目标的行为。 | 系统将记录本次违规情况，一定时间内多次触发时将降低信用评级并临时禁止参与排位赛。 |
| 中途退赛 | 指在游戏中主动放弃比赛退出游戏对局的行为，或因其他原因导致游戏客户端被关闭/无法连接至服务器较长时间的行为。 | 系统将记录本次违规情况，一定时间内多次触发时将降低信用评级并临时禁止参与排位赛。 |
| 干扰他人游戏 | 指在游戏中使用角色行为、道具、场景机制等方式，干扰其他玩家正常游戏体验的行为。 | 系统将记录本次违规情况，一定时间内多次触发时将降低信用评级并临时禁止参与排位赛。 |
| 拒绝确认 | 指在游戏开始时，没有点击“确认进入”导致游戏无法正常开始。 | 短时间内多次触发时，继续尝试匹配时将延时并进入低优先级队列，一定时间内多次触发时将降低信用评级并临时禁止参与排位赛。 |
| 语言违规 | 指在游戏中，通过语音或文字聊天发送含人身攻击、宣扬仇恨、政治敏感、淫秽色情等令人不适的言论。 | 系统将记录本次违规情况，触发此判定时将直接导致信用评级下降。 |
| 操控比赛 | 指通过恶意组队、撞车、收买对手等以控制对局结果的行为。 | 触发此判定时将直接导致信用评级下降。 |
| 第三方软件/外挂 | 指使用第三方软件或恶意程序等技术手段修改游戏客户端表现的行为，若您与有此行为的玩家连续组队，可能同时受到惩罚。 | 触发此判定时将直接导致账号封禁和信用评级下降。 |
| 代练、炸鱼 | 指一定时间内设备登录多个账号且当前对局中的表现远超于当前段位的水平时，会触发代练、炸鱼的判定。 | 触发此判定时将直接导致信用评级下降。 |

## 信用分奖励

等级为【优秀】及【完美】的引航者可以获得相应米雪儿时装/勋章/封装奖励使用权，但信用等级一旦降低将收回使用权。
