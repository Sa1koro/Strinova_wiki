## 角色小传

### 孤独的探险家

|  |
| --- |
| **解锁条件：**达到0好感度解锁 |

从有记忆的年岁开始，心夏就习惯了一个人的家。  
心夏的父母当然并不希望自己的女儿经历这样一个寂寞的童年，但是，在人类向卡丘世界展开大移民之前的那几年，这两位知名的学者不得不奔波辗转于世界各地，记录和保存人类珍贵的文化遗产。这是他们为之献身的事业。  
年幼的心夏还无法理解父母所肩负的重大使命，甚至无法理解“孤独”和“寂寞”——不，她并不感到孤独。  
即使是在小小的家中，也有数不尽的新鲜事物等待着她的探索。  
第一次自己穿衣服，第一次自己洗澡，第一次自己做早餐，还有……第一次把高压锅炸飞。  
每一个第一次都让心夏雀跃不已。  
她就像一个天生的探险家，用纯真的好奇心独自揭开着世界的神秘面纱。  
可是……她唯独没法给自己讲睡前故事。因为讲故事的人自己可不能睡着。  
想到这一点时，年幼的心夏也许是第一次感到了孤独。

### 海兔先生的故事

|  |
| --- |
| **解锁条件：**达到200好感度解锁 |

“心夏，生日快乐！这次的礼物有些不一样，我们不在的时候，可以让它给你讲故事。爱你的爸爸妈妈。”  
每年生日，心夏都会收到父母从远方寄来的信件和礼物。礼物的内容五花八门，奇怪的人偶、精巧的织品……每次心夏拆开礼物之前都会猜测一番，再在下一次视频通话时大加点评，这是他们一家特有的亲子互动。  
但是这一次的礼物，实在超出心夏的预料——那是一个圆滚滚的桶型智能家政机器人。  
在见惯了各种稀奇古怪的古董、文玩之后，这是心夏与现代前沿科技产品的第一次碰撞。  
简洁的外形、坚固的新型材料……当然还有神奇的人工智能，这些现代工艺美学的结晶让心夏感受到了一种完全不同的美。  
按下开机键，机器人头顶的感应器就滴溜溜地转了起来，就像两只兔子耳朵。  
“机器人先生……请问你是兔子吗？你怎么会有一对兔子耳朵？”  
“纠正，本机的‘触角’是模仿海兔设计的生物工学感应器。”  
“所以……不是兔子？是海兔先生吗？”  
“指令不明。请求确认，您是否想要将‘海兔先生’登记为本机的用户昵称。”  
“我听不懂……海兔先生，你能给我讲讲你自己的故事吗？”

### 没有心的玩具

“为无人机找到自己的灵魂”。  
导师的一句话，让心夏陷入了漫长的思索。  
心夏对于机器人工学的兴趣完全源自于她对新鲜事物的好奇心，从来没有想过自己设计和制作的道具要用于什么样的目的。心夏决定要找到一个最适合它的工作。  
轻巧的身体让它能飞上天空，拥有广阔的视野，小巧的体型让它可以穿过缝隙，在复杂的地形中穿行……综合这些优点所得出的结论是——  
“啊啊——还是想不出来！”，苦恼的心夏挠着头在校园里踱步。

说完这句话的少女，很快被不远处一位看似欧泊高管的女性叫走了。  
留在心夏记忆中的，只有绘在少女衣裙上的那片灿烂星空，以及这句无心之语。
