## 角色小传

### 佣兵的自我修养

|  |
| --- |
| **解锁条件：**达到0好感度解锁 |

绯莎从事佣兵工作有多久，她已记不清了。而唯一与她相伴多年的，只有狩猎时使用的枪械与匕首。  
她不记得自己的父母是谁，没有家庭的概念。因为她不需要依靠父母就独自生活到了现在。  
她也不记得是谁传授了她一身战斗本领。也许她曾有一个师傅，亦或许没有。  
这些事情对她来说毫不重要，她也没有兴趣了解或思考。  
有委托了她就便执行委托，并一丝不苟地按照雇主的要求解决猎物。  
除了专业高效，绯莎在保密方面也非常让雇主们放心。她一度成为行会里好评数、订单数最多的佣兵。  
尽管见惯了各种阴暗混乱的场面，但她对人类之间的纠纷恩怨毫无兴趣，也从不过问。  
人类在她眼中只有两类，猎物（对手亦等同于猎物），以及其他人。  
干杀手这一行当，要么成为猎人，要么就沦为猎物。  
所以对于绯莎来说，她最感兴趣的是如何变得更强大的，其他的都不重要。这就是佣兵绯莎的活法。

### 探索之旅

|  |
| --- |
| **解锁条件：**达到200好感度解锁 |

执行任务时，绯莎习惯用代号称呼自己，并按照她的心情随时替换。  
“鲨鱼”是她最有名的一个代号。她的一身本领被乌尔比诺看上之后，她的新雇主们就一直叫她“鲨鱼”。  
“鲨鱼？那是什么？”  
“一种很古老而罕见的海洋生物。”  
“为什么是鲨鱼？”  
“因为你很像鲨鱼啊，你不这么觉得吗？”  
这是绯莎第一次有其他事物产生联系，而她原本的名字“绯莎”在她的印象里是一种植物的名字，为什么这名雇主会说她像鲨鱼呢？  
她莫名地想知道其中的缘由，这算是绯莎为数不多感兴趣的琐事。  
与大多数生物一样，鲨鱼还没有适应新世界的环境，活生生的鲨鱼普通人几乎没有机会见到。  
而绯莎只通过文字、图片或商品了解这种陌生的生物。  
她知道有一种很流行的鲨鱼零食叫甜甜鲨，轻巧便携，能够快速补充能量，这是她除了巧克力和鲷鱼烧之外，最喜欢的食物。  
还有一种鲨鱼形状的整蛊玩具，被称作“咬人鲨”，每次路过玩具店，绯莎都会都驻足研究一番（如果有哪个委托的报酬是咬人鲨玩具，绯莎或许会立刻接下）。  
但真正的鲨鱼是一种尖头尖尾、面目狰狞的海洋生物，她很难将自己与这种遥远而陌生的生物联系起来。  
寻解未果的绯莎决定亲自去问一问那名雇主。

### 向往的平静

|  |
| --- |
| **解锁条件：**达到500好感度解锁 |

“嗯……大概是因为你和鲨鱼的生存方式很像吧。”  
但是，如何相像呢？  
面对绯莎的追问，这名雇主稍微耐心地解释了一番——  
“鲨鱼对‘血’的味道天生敏感，一旦猎物负伤就无法摆脱鲨鱼的追捕。从这个角度来说，你不就是一条鲨鱼么？”  
不，并非如此。鲨鱼是为了自己而狩猎，而绯莎从来都不是，毕竟鲨鱼不需要在凌晨三点被叫起来执行任务。  
绯莎听到这种说法时，心里第一次产生了不快。  
尽管无亲无故，以“狩猎”为生，但绯莎清楚地知道，自己是一个人类，而不是嗜血的怪物。  
在她的内心深处，也有隐藏着对“平静生活”的想象：  
睡觉睡到自然醒，然后把那个喜欢吃薯片的对手约出来对打，直到决胜出谁才是真正的强者；  
或者，把影像店所有的武打片打包租回家，把音量开到最大，吃着鲷鱼烧看上一整天。顺带一提，武打片是绯莎学习战斗美学的重要渠道。  
故而，绯莎追求的“平静”，并非远离战斗，而是让战斗无处不在、充满自己的生活。  
不过，鲨鱼的生活会是什么样的？或许，鲨鱼也有属于自己的“平静生活”？

### 我不是鲨鱼？

|  |
| --- |
| **解锁条件：**达到900好感度解锁 |

“喂，别吃了……我问你，你了解鲨鱼吗？”  
某一次，绯莎主动找到了那个爱吃薯片的对手，不过这回她吃的是仙贝。  
“好像在拉薇的书上看到过，怎么了？”  
“有人说我跟鲨鱼很像……我想知道原因。”  
“所以，你打算怎么做呢？”  
“我想见见鲨鱼，想和它聊聊。”  
“但鲨鱼可不会说话，就像大多数动物一样。而且，我觉得你不像鲨鱼。”  
对方的说法令绯莎倍感震惊，因为一直以来，她都把鲨鱼当作是自己的同伴，是与自己最为相近的“同类”。  
“为、为什么不像……？”  
“因为这个世界的鲨鱼几乎都被关在水族箱里，时间久了，不再接触海洋，它们就会慢慢丢失自己的嗅觉。”  
鲨鱼……丢失了嗅觉？  
“但你不一样，你是嗅觉非常出色的猎人，来去自如。所以，我说你不像鲨鱼。”  
鲨鱼不是同类？难道自己被雇主骗了吗？  
对手的说法令绯莎耿耿于怀，她决定亲自探寻答案。  
听对手说，鲨鱼这样没能适应环境的生物被严格保存在欧泊的管辖之下。  
在一次任务中，绯莎依照雇主的要求秘密潜入欧泊的科研基地——  
不，倒不如说是绯莎主动揽下这个委托，她一直在暗中窥探能够见到鲨鱼的机会。

### 水箱之外

|  |
| --- |
| **解锁条件：**达到1300好感度解锁 |

有情报称，在一个科研基地里秘密保存着一条活的鲨鱼。  
虽然还是在熟悉的凌晨三点，但这次绯莎的工作热情显著提升。她顺着情报给出的坐标，顺利找到了那个困住了她“同类”的水箱——  
隔着冰冷的玻璃，两条“鲨鱼”互相凝望，但对方却在黑暗的水箱里一言不发。  
鲨鱼确实不会说话，对手的第一个说法被印证了。那么嗅觉呢？绯莎迫切地想验证第二个说法。  
但在此时，绯莎的行踪暴露，雇主也给绯莎发来撤离的指令。带着遗憾与一丝丝孤独，绯莎不舍地告别了鲨鱼。  
在完成那次任务之后，绯莎主动找到正在街舞机上扭动的老板，要求对方把那条鲨鱼搞到手。  
为此绯莎愿意拿出自己的全部积蓄，但贵为商业精英的老板却极为罕见地面露难色……  
其他雇主也是一样，没一个人能帮到自己，绯莎对此非常失望。她重复着日复一日的委托，她本以为自己也无法走出水箱……但在某次狩猎中，意外降临了：一个奇怪的人出现，意外地打败了自己，并做出了一个有趣的承诺。  
绯莎无法用已有的两个分类概括这个人，她单独为其开设了一个新的分类。  
在那个承诺中，她看到了一丝实现“平静生活”的希望。  
也许是时候告别凌晨三点被老板叫醒、被强劲音乐打扰的生活了……  
她决定向那个人投递一封求职信。  
在写信的时候，她好像看到，水箱里的鲨鱼正在一点点地向洒满阳光的水面游去。

## 回归信

|  |
| --- |
| **触发条件：**   - 与该角色已签订誓约且切换至别的角色宿舍长时间后再切回 - 超过7天未登录游戏 |

引航者，好像很久都没有收到你的委托了，你应该变得比之前还要强大了吧。  
这段时间，我也一直在坚持修行，尝试着各种各样的强者挑战。  
我在一点点地探索着，观察着，了解着，你所说的，鲨鱼看不到的世界。  
新一届的最强格斗王挑战赛开幕了。那只胖猫又叫上了她的伙伴，向我们鲨鱼小队下达了挑战书。  
  
作为我的格斗游戏教练和同伴，你不会缺席这次对决的吧？  
咬人鲨游戏的训练我也一直在坚持，这次我一定能战胜你。  
还有很多属于强者的挑战，很多全新的风景……我都想和你一起看看。  
鲨鱼不会忘记最重要的同伴，我就在这里等你。
