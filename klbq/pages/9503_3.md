|  |

| **第三赛季——** |
| **暮光幻蝶** |

[![壁纸-第3赛季暮光幻蝶.png](https://patchwiki.biligame.com/images/klbq/thumb/c/cf/atpaqkaja8i5m2h2ruz8cm9mrfsj71w.png/700px-%E5%A3%81%E7%BA%B8-%E7%AC%AC3%E8%B5%9B%E5%AD%A3%E6%9A%AE%E5%85%89%E5%B9%BB%E8%9D%B6.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E5%A3%81%E7%BA%B8-%E7%AC%AC3%E8%B5%9B%E5%AD%A3%E6%9A%AE%E5%85%89%E5%B9%BB%E8%9D%B6.png)

乌尔比诺盟主大选在即，格罗大银行事件成为了拉开这场斗争序幕的导火索。  
  
出现在格罗夫银行的手环与视觎它的神秘组织，诸多谜团的答案指向了被遗忘的过去。  
  
旧日的伤痕，消失的记忆，以及加速变化的世界.....一切，将在这个夜晚燃烧殆尽...
