## 角色小传

### 高洁之誓

|  |
| --- |
| **解锁条件：**达到0好感度解锁 |

“我将高傲不屈，纯洁不朽……”  
年幼的奥黛丽向着纯白的独角兽家徽立下誓言。  
作为一名格罗夫家族的成员，特别是格罗夫当家人的女儿，她从小就被教导，要始终优雅，永葆高洁，不要轻易地屈服低头。  
奥黛丽的童年时光被严苛的父母安排得满满当当，每一天她必须完成十多门课程的学习，包括马术、谈判、金融、音乐、舞蹈……  
她总是能以超乎同龄人的成长速度接纳这些专业知识。  
家族的荣耀与责任或许过于庄严而高大，但在奥黛丽看来，她并不会把这些东西视作压力与负担。  
独自一人的时候，望着手里的徽章，她由衷地希望自己身上也能迸发出高洁的光辉。  
或许是任性，亦或许是作为女儿的小小愿望，在忙完所有的课程之后，她会拉上父母开一场音乐茶话会。这是她对父母的唯一要求。  
咖啡、巧克力，配上小提琴曲，一家人欢聚一堂，这样的茶话会才是完整的。  
当琴弓与琴弦共振齐奏时，动听悠扬的旋律响起，女孩仿佛被一片纯白的光芒笼罩。

### 光影交错

|  |
| --- |
| **解锁条件：**达到200好感度解锁 |

作为休诺·格罗夫的独生女，奥黛丽十多岁的时候就跟着父母参与企业管理。  
格罗夫家族传续数百年，在金融银行、股票证券、汽车制造、能源开发等诸多领域都能瞥见独角兽的身影，其中的利益网络也错综复杂。  
奥黛丽所要面对的是整个商业帝国的荣光与阴暗。  
在商业合作中她被竞争对手反复中伤坑害，这使得她对人与人之间的真心产生质疑，她习惯性地将家族以外的关系排除在真情之外。  
由于早年积攒了大量的管理经验，奥黛丽很早的时候时候就成为了父亲的管理副手。每次家族聚会，父亲都会带奥黛丽出席。  
酒席之上，盛装的少女与那么多同族聚会共餐，虽然她深知人心莫测，但在内心深处，她仍旧对亲情抱有一丝期盼。  
“奥黛丽，成人礼快乐！往后你将会收到一份特别的礼物！”  
奥黛丽的亲叔叔布莱兹·格罗夫如是祝福道。  
奥黛丽与这位长辈碰杯，以示敬意。她带着纯粹美好的愿望期待着那份礼物，  
她却无法预知，在碰撞的酒杯下隐藏着怎样的阴谋。

### 暮色降临

|  |
| --- |
| **解锁条件：**达到500好感度解锁 |

“人心无以预料，亲情坚不可摧。”  
起初奥黛丽与父亲一样坚信，同族的血亲绝对秉持着一样的高洁伟岸。  
外界谣传布莱兹·格罗夫觊觎家主之位已久，奥黛丽不会轻易怀疑自己的亲人，但随后发生的事情远超出她的预料——  
公司的执行部长被暗杀，父母受到人身威胁，公司的营收状况一落千丈。  
随即，以布莱兹为首的格罗夫们纷纷倒戈，他们向竞争对手低价抛售股票，致使格罗夫家的资产大量流失。  
奥黛丽与父亲站出来主持大局，安抚人心，却无人响应。父女二人声嘶力竭的演说换来的只有白眼与嘲笑。反对者们云集，他们发起公投重选当家人，父女二人被无情地驱逐。  
那场公投过后，奥黛丽的心中立下了一堵戒备的高墙，父母之外的人都不可信赖。  
为了留存家族基业，奥黛丽宣布公司退市，并且不断挖掘新的商机。  
此时格罗夫家族早已四分五裂，外界或许认为这会是格罗夫家族的落日。但只有奥黛丽知道，真正的格罗夫从不会轻易低下高昂的头颅。

### 纯白孤独

|  |
| --- |
| **解锁条件：**达到900好感度解锁 |

商业之都纽特朗中林立着瑰丽璀璨的摩天大楼，有朝一日，奥黛丽渴望在这里筑起更加恢弘雄伟的建筑，在上面铭刻上象征格罗夫家族的独角兽。  
她往后的道路注定孤独，尽管有一众家族旧部以报恩为由，愿意追随奥黛丽，奥黛丽却无法相信他们。  
进军纽特朗后，奥黛丽迅速瞄准了高端商城领域，她深入发掘，并与强大的商业联盟“乌尔比诺”建立合作。很快，新生的格罗夫在纽特朗立稳根基。  
奥黛丽与同族只保持着商业上的往来。居高临下，冷若冰霜，令人望而生畏，这是家族成员对该时期奥黛丽的全部评价。  
尽管如此，格罗夫们无一不认可奥黛丽的才能，关于新一任家主的人选，奥黛丽的呼声达到了顶峰。回想起父亲的遭遇，奥黛丽全然不在乎他们的拥护与支持。  
然而，长久高压地操盘商业以及对旁人设防，使得她的身心受到了极大损耗，她终于还是支撑不住病倒了。  
在被送入医院时，她最后睁眼看到的有泪眼婆娑的父母，还有一群陌生的同族。  
她在内心深处对双亲呐喊，不要相信那些人，那些人从不值得信任……  
随着她身体的状态抵达了极限，她闭上了双目。

### 林中困兽

|  |
| --- |
| **解锁条件：**达到1300好感度解锁 |

悠扬的音符温暖地包裹着她，失去全部记忆的少女从一片黑暗混沌中醒来。  
一对夫妇坐在自己周身，远处一个同龄的女孩演奏着小提琴曲。但那些遥远而又温暖的音符却又马上消散了。  
“你们是什么人？”懵懂中的少女发问。  
“我们……都是你的家人。”那对夫妇回答道。  
她的病房外站满了很多人，他们的胸前佩戴着一枚独角兽勋章。这似乎是一个大家族？  
随身听、骨瓷制咖啡杯、《话语的螺旋角》精装版……听说这些都是“奥黛丽”最珍爱的东西。  
她带着这些与“奥黛丽”相关的旧物走访亲友，不断去寻找与“奥黛丽”相关的记忆碎片。这趟旅途非常漫长，但她坚持了下来。  
复苏后的她比此前更加投入工作，她主动肩负起守护整个家族的责任，甚至决定参选下一任乌尔比诺的盟主。  
繁忙的事务又回到她的身边，但这次似乎有所不同，家族内部对于她的回归议论纷纷。  
破碎的镜子难以复原，收集记忆再次拼凑出的“奥黛丽”还是原来的那个她吗？  
她感觉自己如同童话故事中的困兽，在孤独的森林里，没有任何一个同伴……  
如果能够追逐那温暖的音符，走出森林……少女将这个小小的心愿埋藏在心底。
