## 角色小传

### 看不见的情绪

|  |
| --- |
| **解锁条件：**达到0好感度解锁 |

在大众的眼中，香奈美的出身优越，她的父亲是行业里赫赫有名的理论物理学家，母亲则是出版了畅销小说的作家。香奈美在温馨开明的家庭氛围中成长，很早就显露出了舞台表演的天赋。  
然而现实却是千差万别的，香奈美在家里从未获得过足够的关注。称之为父亲的男人只专注于研究的课题，称之为母亲的女人郁郁寡欢于自己写不出好作品，就连年长的哥哥也不过是在用成绩来博得父母的一点关爱。  
年幼的香奈美经常会被托付给领居或保姆，她只有在家庭聚餐日，才有机会与父母亲近。他们之间的情感仿佛是凝滞的，父母显然对于养育孩子感到无法适从。香奈美很自然地认为自己越懂事，父母就会越高兴，于是她变得很独立，尽量不给家里人添麻烦，就像是一个小大人。  
“好像没有人会关注我……”独处的时候，香奈美经常会发出这样的疑问。  
12岁那年，香奈美在网上偶然间看到了偶像练习生的选拔活动，她目不转睛地盯屏幕，站在聚光灯下的少女们被人群围绕着，那一句“抓住这次机会，让所有人看到你”一下子就击中了香奈美的心。  
香奈美认真地填写了报名表，期盼着中选，只可惜几个月过去了，她没有收到任何回信。

### 寻求关注的本能

|  |
| --- |
| **解锁条件：**达到200好感度解锁 |

直到某天放学回家的路上，一名来自艾顿娱乐的星探小姐找到了香奈美，她邀请香奈美加入偶像预备营。  
此时香奈美并没有马上答应，她鼓起勇气问星探：“我明明已经落选了，为什么你还会来找我呢？”  
星探卡蒂大笑着回答：“我恰巧看到了你之前的报名表，大部分人在偶像宣言那一栏只写了一两句话，而你却整整写满了一页。所以我猜想你应该比任何人都更渴望站上舞台吧？”  
被说中心事的香奈美当即就羞红着脸逃开了，但是没过几天，她还是忍不住拨通了名片上的电话。至于后面的故事就很顺理成章了，她签约成为了练习生，开始了长达三年的艺能训练。  
在这期间，有好多女孩为了能早点出道采用了组团的形式来争取机会，然而没有人邀请过香奈美。香奈美向已经升为主管的卡蒂请教原因，卡蒂直白地告诉她：“因为你太过于耀眼了，谁和你组团只会被沦为伴舞。”这是香奈美人生中第一次获得如此高的肯定，她愈发坚定了成为偶像的梦想。  
15岁那年，香奈美参加了节目《公开试镜》，她凭借着“天真烂漫”的笑容迅速走红，最终以第一名的成绩出道。香奈美坐在了艾顿城上空的秋千上，对未来的演艺生涯有着无限的憧憬。

### 成名之路

|  |
| --- |
| **解锁条件：**达到500好感度解锁 |

人们的关注是非常具有时效性的，这是香奈美进入演艺界学到的第一课。  
《公开试镜》节目一播完，香奈美的热度就迅速降下去了。她即将要面对的是极为残酷的业界，和没有工作邀约的困境。  
为了能让香奈美闯出一条道路，卡蒂担任了香奈美的经纪人，为她组建了运营团队，还找来了实力派歌手夏露担任香奈美的声乐老师。  
出道的第一年，香奈美过得异常艰辛。她除了要接受高强度的唱跳训练之外，还会在经纪公司的安排下，去跑各种综艺节目的通告、出席小型商演活动、客串微电影和音乐MV中的小角色……  
随着香奈美混迹在娱乐圈的时间越来越长，她已经能清晰地辨识复杂的人际关系，用一套灵活的方式去应对各种身份的人了。即使对方怀有敌意，她也能用笑容去轻松化解。  
经过了整整两年的厚积薄发，香奈美的国民认知度终于打开了，她的迷你专辑也迎来了大火。特别是歌曲《心海》连续三个月都蝉联音乐网站人气歌曲排行榜的冠军，获得了钻石唱片等奖项。  
香奈美被誉为甜美少女的代言人，闪耀在舞台上的一番星，艾顿娱乐公司也获得了足够的资金来为她举办巡回演唱会。

### 过去的影子

|  |
| --- |
| **解锁条件：**达到900好感度解锁 |

“初始”巡回演出期间发生过一个插曲，某次香奈美在进行跨城穿梭时，发生了由于飞行器系统故障导致的事故，使她迫降在了未知区域。附近的普雷顿管理中心在接收到香奈美的紧急呼救信号后，派出了救援队将她安置到了柯西街区。  
香奈美在柯西街区见到了接待她的明，明表示自己是第一次见到偶像，心情特别激动。明热情地让香奈美住进了自家的客房，还带她去大吃了一顿。  
香奈美以前了解到的普雷顿是由边缘人组建的垃圾城区，那些不思进取的人们企图与欧泊政府叫板，成立了一个极具威胁性的剪刀手组织……然而就香奈美的亲身体验来说，她感受到的是居民们的朴实，人与人相处的温情。  
距离经纪公司专机抵达只剩三天的时候，香奈美自发地在普雷顿举办了小型义演。在演出的过程中，香奈美惊奇地发现台下有些人流泪了。事后，香奈美向明询问原因。明告诉香奈美那些人已经被欧泊的轮回计划抹去记忆了，流泪只是因为他们被歌声唤醒了一点零星的画面，可是谁又会为他们发声呢？  
香奈美想起了曾经，想起了自己不被任何人关注的时光。她一时冲动跟明说道：“如果有需要，我也会成为你们的志愿者……”。

### 矛盾的心理

|  |
| --- |
| **解锁条件：**达到1300好感度解锁 |

“初始”巡回演唱会圆满落幕了，香奈美也回到了艾顿城，继续依照顶流偶像的轨迹去工作和生活。  
出乎香奈美意料的是明被调到了主城区执行任务，而且明会经常主动联系香奈美，香奈美也帮着明去打探过一些消息。这样一来二去，香奈美友好的态度让明误以为她真的愿意追随剪刀手组织。而实际上，香奈美处于一种被动答应的状态，但是她太擅长将这种矛盾的心理掩盖过去了。  
作为曾经的透明人，香奈美对于普雷顿人们的事情无法坐视不管，但是她清楚地知道以自己的身份绝不能参与这种危险的事情。她多么希望可以出现一个人帮自己理清杂乱的思绪，为她找到处理问题的答案。  
在一次爆破行动中，香奈美受了伤，剪刀手组织暂停了她后续的任务安排，让她好好休养一段时间再提交归队申请。  
香奈美不知道这件事对她来说是幸运还是不幸，因为她本身就很迷茫，她难以把控自己的选择和生活。就连当初她被视为志愿者，由拉薇领着去挑选武器的时候，她也经历了内心的反复纠结，最终缓慢地拿起了狙击枪。在香奈美看来，狙击手的身份隐匿，待在不易被察觉的暗处，可以最快地转移位置……所以算是可行的吧。  
我是谁？我在干什么？  
香奈美需要先了解自己，才有可能真正明白战斗的意义。
