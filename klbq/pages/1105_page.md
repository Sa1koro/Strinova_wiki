## 武器数据

|  |  |
| --- | --- |
| 名称 | 减速雷 |
| 种类 | 战术道具 |
| 获得方式 | 账号等级14级解锁 |
| 介绍 | 在投出5s或碰撞三次后激活，炸开一大滩颜料，在其持续时间内使范围内的敌人被大幅减速并禁止弦化。 持续时间为7秒，影响半径为6米。 减速效果为30％，不可叠加，且与多数附带减速效果的技能共享状态栏。 烟雾弹，减速雷，风场雷的运动轨迹完全相同。 |
