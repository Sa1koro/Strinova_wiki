  **在6V6刀战模式下，每把近战武器有着不同的攻击效果，可按需选择。**
- [![](https://patchwiki.biligame.com/images/klbq/thumb/3/35/mn2fdw5laga72ag5bcdkdd7izvesqfv.png/889px-%E7%8E%A9%E6%B3%95%E6%95%99%E5%AD%A6-109002.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E7%8E%A9%E6%B3%95%E6%95%99%E5%AD%A6-109002.png)

  **能量槽充满后，超弦体可进入强化形态：移速增加，伤害减免，重击的冷却时间减半。**
- [![](https://patchwiki.biligame.com/images/klbq/thumb/9/9f/0n0036ceqr4f3jrl402eniidzr7bpw3.png/889px-%E7%8E%A9%E6%B3%95%E6%95%99%E5%AD%A6-109004.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E7%8E%A9%E6%B3%95%E6%95%99%E5%AD%A6-109004.png)

  **击败人数最快达到指定上限的队伍获胜。**

## 获胜条件

- 规定时间内击败指定数量的敌人。
- 比赛结束时分数更高。

## 模式设定

- 模式地图：[莱特园区](./6484_page.md "莱特园区")、[2号仓库](./6581_2.md "2号仓库")、[码头小镇](./12436_page.md "码头小镇")、[花明弓道馆](./6810_page.md "花明弓道馆")
- 仅可使用近战武器，每把近战武器可产生不同的攻击效果，武器可在战斗途中使用B切换。
- 可使用R【格挡】抵御敌人的重击，成功格挡后会获得伤害提升，【格挡】有1秒的冷却时间。
- 可通过击败对手、造成伤害、格挡等行为来积蓄能量。
- 所有超弦体默认均为130生命值+80不可回复轻甲。
- 能量槽充盈完毕后，即可启动G【极限爆发】，短暂强化自身属性。
- 击败人数最快达到指定上限的队伍获胜。

## 模式武器

### 忍锋

|  |  |
| --- | --- |
| [武器 15202001.png](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E6%AD%A6%E5%99%A8_15202001.png) | 不知名的武器画家所制作，很有格调的武士刀。 |

- **轻击：**快速对敌人施加斩击，造成一定的伤害，拥有较远的攻击距离。轻击不会被格挡抵消。
  - 伤害为70点
- **重击：**在短暂延迟后释放斩击，对命中的敌人造成高额伤害。
  - 可在移动中释放，伤害为210点，可直接斩杀满血敌人，冷却时间为3秒；极限爆发状态下冷却时间为2秒。

### 我要竞鸡

|  |  |
| --- | --- |
| [武器 15203001.png](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E6%AD%A6%E5%99%A8_15203001.png) | 这柄独特的兵刃会让持有者涌起莫名的感悟：或许武器的强大不在于本身的锋利，而是一往无前燃烧自己的呐喊与决心。  【“全体单位注意！他们亮出了竞鸡！”】 |

  - 可在移动中释放，伤害为140点，可进行AOE攻击，没有距离和穿透衰减，冷却时间为3秒；极限爆发状态下冷却时间为2秒。
