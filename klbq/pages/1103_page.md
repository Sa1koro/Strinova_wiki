## 武器数据

|  |  |
| --- | --- |
| 名称 | 闪光弹 |
| 种类 | 战术道具 |
| 获得方式 | 账号等级6级解锁 |
| 介绍 | 在投出后1.5秒后引爆，释放强光，引爆时将对40米范围内看到它引爆的敌人和自己造成暂时致盲并干扰听觉，致盲时间随距离递减。 背身面对闪光弹将大幅减少致盲时间，即使在脚下引爆并被看见。 闪光弹以玩家的摄像机视角为影响基准，即摄像机视角内没有闪光弹则影响降低或不会受到影响。 |
