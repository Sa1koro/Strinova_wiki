### 基本规格

[![M1887.jpg](https://patchwiki.biligame.com/images/klbq/thumb/4/44/inmo4y6cdawku15ta3wszvrh8kw34ow.jpg/500px-M1887.jpg)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:M1887.jpg)

|  |  |
| --- | --- |
| 重量 | 空枪（标准型）：3.6千克（8磅） |
| 长度 | 997毫米（39.25英寸） |
| 枪管长度 | 508毫米（20英吋） |
| 子弹 | 12铅径霰弹 |
| 口径 | 18.53×69.85毫米（0.73×2.75英吋） |
| 枪管 | 1根，滑膛，无枪口装置 |
| 枪机 | 杠杆式 |
| 发射模式 | 连发 |
| 有效射程 | 50公尺（54.68码，164.04英尺） |
| 供弹方式 | 管状弹仓，5+1发 |

作为较少见的杠杆动作原理的霰弹枪，国内的枪迷可能对第一支真正成功生产和销售的连发霰弹枪——温彻斯特1887型没什么印象。但如果提起《终结者2》（Terminator 2）里面州长大人骑摩托车拉风必备的枪，可能很多人都会对阿诺德·施瓦辛格单手上弹、开枪的耍酷动作记忆尤新。这把枪是由著名的枪械设计师约翰·勃朗宁设计，由温彻斯特连发轻武器公司（Winchester Repeating Arms Company）生产的。

在1880年代初期，温彻斯特要求约翰·勃朗宁为他们设计一支12号口径的连发霰弹枪，虽然勃朗宁主张采用泵动式原理，但温彻斯特公司则认为该枪最好采用杠杆动作原理，因为在当时这是温彻斯特公司的招牌设计，温彻斯特几乎成了杠杆连发枪的别名。勃朗宁最后设计出来的枪是一支发射12口径霰弹、容弹量6发的杠杆连发霰弹枪。其实它的管状弹仓中只能装4发霰弹，加上弹膛中的1发和供弹机构中的1发，合共有6发容量，有些计划方法不算上弹膛中的1发，所以认为是5发。

在当时，所用的霰弹还是以黑火药作为发射药的，而且温彻斯特1887型霰弹枪的弹膛只能容纳使用黑火药的12号霰弹。在1890年代后期无烟发射药开始普及后，温彻斯特公司把1887型霰弹枪重新设计成适合无烟发射药霰弹型号，重新命名为1901型，但1901型发射的是10号无烟发射药霰弹。之所以不推出12号口径无烟发射药霰弹的1887型，是因为温彻斯特不想让1901型与当时更受欢迎的12号口径1897型泵动霰弹枪争市场。

事实上在温彻斯特1897型和其他同时代的泵动霰弹枪推出后，1887/1901型杠杆连发霰弹枪就没什么市场了。从1887年至1901年间，1887型霰弹枪共生产了64,855支，而从1901年至1920年间生产了79,455支1901型霰弹枪。

多年来，一些枪械生产商也试着生产发射现代霰弹的1887/1901型霰弹枪，以便让一些喜欢玩牛仔枪的人不必费尽心思去找黑火药霰弹。不过这些枪的数量都很少。最近，有两家外国公司也计划大量仿制现代口径型的1887型霰弹枪，其中澳大利亚的ADI有限公司已经试产成功，但由于销售利润的分配问题，一直没能正式上市。而中国北方工业公司推出的发射12号2 ¾英寸弹的1887型霰弹枪（Norinco YL-1887）已经成功出口到美国、加拿大和澳大利亚。

## 更新改动历史

2026年3月10日更新

- 主动技能护盾收起后不会立即开始恢复护盾，而是在2秒延迟后开始恢复
- 冲撞命中敌人后的缴械与沉默持续时间由1秒调整为0.5秒（晶源感染模式中仍为1秒）
- 狙击枪类武器（空境、谢幕曲）会对汐的护盾造成25%额外伤害
- 弦能增幅网络成长项目【独头弹模式】升级后的属性调整如下：

:   :   [![公告-20260310公告11.png](https://patchwiki.biligame.com/images/klbq/thumb/9/94/5fxwlpb7rhy5w6m44xdwl6xcuica9nd.png/500px-%E5%85%AC%E5%91%8A-20260310%E5%85%AC%E5%91%8A11.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E5%85%AC%E5%91%8A-20260310%E5%85%AC%E5%91%8A11.png)

- 弦能增幅网络觉醒2造成伤害由60调整为50，引爆护盾后会返还20%护盾冷却并获得持续1秒的30临时护甲
- 弦能增幅网络觉醒3激活后汐自身转移至护盾的伤害由115%调整为125%
- 修复了【汐】在【枪王乱斗】模式中，持有其他超弦体枪械时，无法正常切换开火模式/准镜倍率的问题。

晶源感染模式调整：

- 主动技能护盾由1600调整为2000，护盾恢复速度调整为200每秒
- 护盾被击碎前会由护盾承受伤害(激活觉醒2不会导致该效果失效)
