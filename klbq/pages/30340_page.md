## 角色小传

### 纯真的调色盘

|  |
| --- |
| **解锁条件：**达到0好感度解锁 |

在玛德蕾娜·利里7岁生日的那天，身为艺术家的父母为她订制了一块调色盘。  
这是她生平收到的第一份生日礼物，小玛德蕾娜给它起名为“七彩”，幻想着能在里面填上无穷无尽的色彩。  
她用“七彩”创作了一幅画作。平日严肃的父母望着那幅画，罕见地面露幸福的神情——玛德蕾娜很喜欢那样的表情。  
她决定，往后一定要加倍努力地练习绘画，她想更多地带给别人快乐。  
按照利里夫妇的安排，小玛德蕾娜和姐姐加拉蒂亚每天都要进行大量的练习，从早上6点一直持续到晚上9点。  
那块调色盘渐渐地从白白净净变成五颜六色的，而她的脸蛋或者手心上也经常沾上颜料，像一只脏兮兮的大花猫。  
与之相反，她的姐姐很不满于这种既定的生活，白天要么玩消失，要么带回蠕动着的奇怪“素材”；然后夜里姐姐的身影再如同鬼魅般突然浮现。这一系列行为带给了同一屋檐下的玛德蕾娜诸多心惊肉跳的回忆。  
即使如此，玛德蕾娜还是主动帮忙给姐姐打圆场，躲避父母的盘问。  
因而，她也养成了乐于助人的好习惯，外出取材时顺手搞定邻居们的委托，使她在街坊领里之间收获了很高的评价。  
带给别人快乐，似乎成为这个天真小姑娘的幸福感来源。

### 陌生的天花板

|  |
| --- |
| **解锁条件：**达到200好感度解锁 |

明星院校“伊莱特隆艺术学院”培育了无数艺界英才，入读这所院校也是利里夫妇对姐妹二人的最大期望。  
看到姐姐以保送免试的形式轻松入学，玛德蕾娜心里暗下决心，一定要考上那所学校！  
之后她参加了号称“炼狱级难度”但通过率在报考生中极有信服力的封闭式集训。  
往后回忆起那段时光，她只记得她累到差点把颜料当早饭下咽，做梦都在刻画光影、塑造形体。  
所幸最终她以齐平录取分数的成绩勉强考入了那所学校。  
在父母送别她的那天，她却觉得心里空落落的，这是她第一次离家这么远修习艺术。  
特别地，她带上了那块调色盘同自己一起远行。  
往后全新而陌生的学校生活令玛德蕾娜产生诸多了不适。  
素描课上交了油画课的作业，穿着雕塑课的衣服来上游泳课，诸如此类的迷糊事情时有发生。  
“天才加拉蒂亚的妹妹居然是个迷糊鬼”——这种印象与人们的预期严重不符。  
玛德蕾娜的部分迷糊事迹，不经意间成为学生们的趣谈，她以“某某的迷糊妹妹”的名号在学生间出名。  
每当结束快节奏而混乱的一天后，疲惫的玛德蕾娜就会躲在宿舍里，望着陌生的天花板，久久地发呆。

### 隔阂

|  |
| --- |
| **解锁条件：**达到500好感度解锁 |

绰号也好，流言也罢，玛德蕾娜对学校里的一切习惯性地一笑置之。  
她将更多的精力投入到学业中，艰难地保持着中等偏上的成绩。  
临近毕业设计之时，她提出的多个方案被导师否决，于是她向早已毕业的姐姐加拉蒂亚寻求帮助。  
玛德蕾娜给姐姐展示了自己精心设计的三十多张草图，她期望着姐姐能够理解与欣赏她的设计。  
但是姐姐的反应却稀松平常，淡淡地说了一句——  
“你的设计根本算不上是艺术。”  
姐姐说话向来单刀直入，但这句无心之言还是刺痛了玛德蕾娜。  
玛德蕾娜当即要求姐姐展示她的设计，对着那些潦草随性、意味不明的线条圆圈（甚至当中有的是空白图案），玛德蕾娜的心里写满了问号。  
“你的意思是……这些就算是艺术咯？”  
“不然呢？是你那些无人关注的草稿吗？你知道拍卖行给我的估值是多少个理想币吗？”  
“姐姐，艺术是不能完全用金钱来衡量的……”  
“你的想法还真是稚嫩，所以至今为止你还像雏鸟一样蜷缩在被安排好的鸟窝里，连艺术的门槛都够不到！”  
氛围紧张到了前所未有的程度，姐妹二人的争吵吸引了利里夫妇的介入。  
这场争吵最终是如何收尾的，玛德蕾娜早已不记得。  
她和姐姐还保持着从前的和睦，姐姐会给她分享好看的帽子，她会吐槽姐姐古怪的设计。  
然而，从那次争吵之后，玛德蕾娜发现，姐姐在家的时间少了很多，经常神秘消失或失联。  
她不知道姐姐去了哪里，她只是隐隐察觉，有一道无形的壁垒堵在了姐姐和她，和这个家之间。

### 漂流的水滴

|  |
| --- |
| **解锁条件：**达到900好感度解锁 |

毕业后的数年内，玛德蕾娜忙于在各艺术馆、杂志社之间奔走，她寄希望于通过投稿的形式让自己的艺术获得外界的关注。  
而同一时期，她的姐姐，加拉蒂亚·利里，已经登上了乌尔比诺最知名的艺术鉴赏节目《达·芬奇之手》，在闪烁的聚光灯下，展现着迷人的身姿以及她犀利的观点。  
不爱观看这类节目的玛德蕾娜，也从身边人的嘴里得知了姐姐声名远扬的事情。  
尽管她从未停止奔跑的步伐，但是她得到的反响却相对平淡。在姐姐也参与的设计比赛中，她也无一例外地都输给了姐姐。  
风曳镇即兴创作比赛、伊莱特隆艺术学院校友赛……这些败北，几乎多得数不过来。  
“你像雏鸟一样蜷缩在鸟窝里”、“你连艺术的门槛都够不到”……  
姐姐这些聒噪的话语如同回响一样萦绕在玛德蕾娜的耳畔。  
玛德蕾娜觉得在姐姐的光芒之下，自己黯淡得就像发不出光的星星。  
沮丧与迷茫如同一粒粒沙子在她的内心积压，父母问候的电话她开始不知道如何回应，渐渐地她不敢再回到熟悉的家中。  
在没有告知父母的情况下，玛德蕾娜悄悄地决定成为一名自由职业画师，继续在艺术的浪潮中漂流，在山野，森林，溪流边捕捉灵感。  
她带着“设计师不是在创作，就是在取材的路上”这样的理念四处奔走。  
据说在卡丘世界的各个角落都能看到她的身影。

### 缺失的色彩

|  |
| --- |
| **解锁条件：**达到1300好感度解锁 |

玛德蕾娜的艺术长河并非一直毫无波澜，她的“灵光一现”偶尔也能激起几朵水花——  
只是它们无一例外地都是玛德蕾娜因祸得福或助人为乐时的即兴创作。  
诸如，用打碎的玻璃拼装成的《莎丽娜蒙》装饰画，帮邻居家剪的“比萨斜塔”窗花。  
这些设计有的在玛德蕾娜的邻居间掀起一股预订热潮，有的还入选了艺术馆创意设计区的特别珍藏。  
凭借着偶得的好运气，她揽下了不少订单，基本能够保障自己自由画师的生活。  
同时她也享受着，在充满麻烦的日常里，通过帮助别人带来的小小乐趣。  
但是，家里一通通催促的电话紧接而来，她抵不过一连串问询，在父母的建议之下，到白芸艺术馆接手了一份初级画师的工作。  
（顺带一提，这家艺术馆似乎是乌尔比诺某个白姓大人物的私人资产。）  
每天的工作均是一些基础而繁琐的事务，以她的能力应对绰绰有余。  
出于工作的关系，玛德蕾娜经常接触到一些艺术鉴赏相关的资讯或节目，聚光灯下姐姐的光辉又再度映入眼帘。  
“如果我也能身披那样的光芒……”  
怀揣着小小的不甘，玛德蕾娜再次看向了那块污迹斑斑的“七彩”调色盘，假想着里面填满了属于自己的色彩……
