## 角色小传

### 旧时竹影

|  |
| --- |
| **解锁条件：**达到0好感度解锁 |

拉薇年幼时就跟随父亲修习武道，继承了父亲的流派衣钵。  
她十年如一日地在武馆庭院中磨练刀术，院中的竹子已然成为她最熟悉的伙伴。每一株竹子上面刻满拉薇挥刀的痕迹。从它们还是竹苗时，拉薇就对着它们练刀。  
长大后，拉薇的刀愈发凌厉如风，一呼一吸之间，被平整切开的竹叶如雨般落下。拉薇的身手令无数门徒称叹，甚至诸多师兄师姐也不是她的对手。  
尽管如此，拉薇也不甘满足，她永不停歇地打磨自己。或许对于她来说，她本身就是一柄刀。  
旧地球时期的拉薇醉心武道，沉浸在只有刀光与竹影的世界中，唯独和父母保持日常交流，由此她逐渐形成了沉稳内敛、寡言少语的性格。  
“总是形单影只的可怜孩子”、“以刀为友的古怪武痴”，这是武馆同门对少年时期拉薇的最常见评价。拉薇并不在意这些声音，当时的她只听得见竹叶的沙沙低语。  
风又吹了起来，庭院里竹影摇曳，且听风吟，挥刀斩破！

### 暖阳

|  |
| --- |
| **解锁条件：**达到200好感度解锁 |

旧世界的灾难全面袭来，拉薇家的武馆由于长年无人光顾而倒闭，门人们各自逃难。而她的父母因为扛不住那年最冷的寒潮而长眠地下。  
拉薇孤身一人在末世中漂泊了很久很久，她不知道这样的生活还将维系多久……  
“卡拉彼丘将是人类栖居的新家园，那里蕴含着无限可能，不再有贫困疾苦，不再有分裂剥削……”残破的街巷反复播放着这样的广播。
