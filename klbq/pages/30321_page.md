## 角色小传

### 玩家的专属作派

|  |
| --- |
| **解锁条件：**达到0好感度解锁 |

“游戏伙伴，说好了，一定要平安回来啊~”  
“嗯，玩家之间从不爽约！”  
每次搜查官乔治·李外出执行任务时，父女之间都会进行这样的告别。  
玩家之间约定好的事情，不能反悔，不能缺席。  
每次外出执勤，父亲都会遵守约定准时回来，陪伴女儿体验她最爱的游戏——《NO.1的正义英雄》。坐在家庭电视前相约着玩游戏，是父女之间为数不多的共处时光。  
父亲操控主角“火力大喵”，女儿配合操作主角的搭档“喵喵卫士”，父女二人组共同对战邪恶的魔龙。  
想要以最快的速度通关游戏，往往需要不惧险阻的勇气、敢于突破的创新精神，这一点逐渐形成了米雪儿特有的“玩家思维”，由此，她成长为一名游戏好手。  
她会不自觉地把这种思维代入到现实生活中，做事风格大胆跳脱。  
开明的父亲非常支持女儿的做法，他鼓励米雪儿勇敢尝试任何她想做的事情。  
“遵循自己的内心，勇敢无畏地尝试，这才是玩家的风格。”

### 小小的心愿萌芽

|  |
| --- |
| **解锁条件：**达到200好感度解锁 |

在《NO.1的正义英雄》里，两位主人公时常需要挺身而出，同被魔龙蛊惑的怪盗团伙、亡命恶徒斗智斗勇。  
在这些对战中，米雪儿遭遇到的“变装怪盗”、“电锯连环杀手”、“怪人弗兰肯斯坦”都有一种无比熟悉的感觉。这些反派的原型似乎都是父亲已查处捕获的要犯。  
在某个卡关的下午，米雪儿在“NO.1的游戏论坛”查询攻略时，发现了一个讨论主角“大喵”原型的热帖。  
这个热帖极其严谨地论证了“大喵”的原型正是破获了无数悬案疑案的搜查官——乔治·李。论坛里的游戏玩家无一不在称叹这位传奇一般的搜查官英雄。  
在那一刻，米雪儿的内心深处萌发了此生最强烈的愿望。  
在她16岁生日的当晚，父亲一如既往从任务中归来，米雪儿望着那身帅气的制服与闪耀的徽章，同父亲约定：“我要成为和你一样的正义英雄！”  
父亲像从前那样轻抚她的额头，不经意间发现，这个小丫头似乎已经长大了。

### 新世界的搜查官

|  |
| --- |
| **解锁条件：**达到500好感度解锁 |

人类开启了新维度空间，在新世界政府的邀请下，父亲带着米雪儿一同跨越维度去往新世界，加入了欧泊。  
从搜查官学院毕业后，米雪儿报名了欧泊搜查官考试。实战考试期间，她不按常理出牌，频频出人意料。  
她以最快的速度完成了考试，但中途屡次遭到监考官的黄牌警告。  
沮丧的米雪儿以为自己没有通过考试，回家的时候脏兮兮的小脸像只花猫。  
就在她以为自己无缘英雄梦时，父亲告知了她通过考试的消息，并为她送上了一份特殊的礼物——两只战术机器猫。  
它们是父亲曾经的侦查机器人，经过战术升级后，成米雪儿的战术伙伴。米雪儿分别给它们起名为“火力大喵”和“喵喵卫士”。  
米雪儿活用游戏中汲取的灵感经验，举动大胆出格，在同一批新人中非常“出名”。她的“玩家思维”也让她显得格格不入，她经常因为违反纪律被人训斥排挤。  
“英雄的女儿也不过如此……”  
“不守纪律的搜查官真差劲！”  
质疑与讽刺之声如潮水般涌来，刚成为搜查官不久，一粒自我怀疑的小小种子在少女的心中种下。

### 不曾断绝的大雨

|  |
| --- |
| **解锁条件：**达到900好感度解锁 |

“这一次大概要很久以后才会回来……”  
“很久是多久？……无论如何，我等你回来！”  
这是父亲崩溃以前，父女间最后的道别。那天下起了大雨，父亲走得很匆忙，神色严峻。  
与往常不同，他没有许诺回来的时间。接连数日的暴雨笼罩了埃利蒙德，远方久久没有传来关于父亲的消息。  
不知又过了多少天，暴雨稍稍停歇，米雪儿收到了一通来自搜查科的电话，对方转告了父亲遭人所害而罹患崩溃症的消息。紧接着父亲的职务被搜查科暂停。  
米雪儿始终坚信，父亲是因为伤病才被暂停职务，但是关于父亲非议的杂音不绝于耳。  
如同《NO.1的正义英雄》里所演绎的，英雄未能实现正义就倒下了，大喵将正义的光之剑转交到了最好的同伴“喵喵卫士”手中。  
父亲在女儿的耳前嘱托了几句，随后少女默默地目送着父亲进入轮回舱。  
那扇大门关上之后，父亲的意志也跟着消失了。一直元气阳光的少女脸上，仿佛有雨滴滑落……

### 阳光总会来临

|  |
| --- |
| **解锁条件：**达到1300好感度解锁 |

对自己的嘲讽质疑、对父亲的歪曲污蔑，这些嘈杂的声音从未消停。  
那场带走父亲意志的暴雨似乎也未曾停歇。  
女孩在无助与迷茫中久久彷徨。  
每每陷入迷惘中，米雪儿总会躲在自己的“秘密基地”。在一个普通的雨天，猫咖店没有开张。米雪儿独自在街头徘徊。  
一声绵软的猫叫声响起，是那两只机器猫咪在呼唤自己。出于担心，两只机器猫咪在冰雨中寻找它们久出未归的主人，因电量将要耗尽而瘫倒在地。
