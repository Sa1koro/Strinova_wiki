### 幽灵熊和技术宅

|  |
| --- |
| **解锁条件：**达到0好感度解锁 |

在卡丘世界的技术宅群体中，流传着一个神秘传说——午夜徘徊的幽灵熊。  
据说，如果选择午夜时分在社区内发布需求，有很大概率被一个头像是“熊”的匿名客接取。  
匿名熊报价低、效率高、成果好，除了话少外，就挑不出什么毛病了。  
不少委托者想把这位民间大神招入麾下，可无论他们开出多高的报价，匿名熊每次都直接拒绝，回复从不超过五个字。  
慢慢的，它因为孤僻独行，被其他人称为“午夜徘徊的幽灵熊”。  
而在卡丘世界的某小区内，也流传着一个神秘传说——X栋X0X的住户是一个从不出门的死宅，几年来都没有人亲眼见过TA，连外卖都是靠一只凶巴巴的机器熊代取的。  
“……这人真奇怪，该不会跟那只熊一样，样子太过凶狠，所以不敢以真面目示人吧？”  
虽然已经尽可能快地把门关上，邻居的议论声还是飘了进来。  
菲用毛绒绒的身体挡住了门缝，似乎想把那些伤人话语挡在门外。  
伊薇特接过菲递过来的便当外卖，踮起脚尖，摸了摸菲的头。  
“菲，不凶狠。”  
“伊薇特……也不奇怪。”

### 最冷的一天

|  |
| --- |
| **解锁条件：**达到200好感度解锁 |

“午夜徘徊的幽灵熊”和“深居简出的死宅”，真实身份是同一个——身边陪伴着机器熊的少女伊薇特。  
伊薇特的童年在研究所里度过。身为高级研究员的父母工作忙碌，无暇照顾她，便把她安置在实验室的角落，丢两本书给她打发时间。  
看似孤独，伊薇特却很享受这种生活：安静，没有人打扰，她可以全身心投入到那些艰涩的研究中，幻想着自己长大后，也能成为在数据台前操作的人。  
她原本以为，这样平静的生活会一直持续下去，直到那一天，她醒来后发现家里安静得有些诡异。  
没有说话声，没有脚步声，没有呼吸声……  
伊薇特从不觉得自己是一个直觉性很强的人，但她固执地认为，彼时冲上心头的笃定预感才是她和父母间亲子关系的最有力证明。  
她不用等待，也不用向谁求证，只要看到书架上一并消失的研究手册，就能确认这个事实——父母离开了，没有带上她。  
伊薇特从小就不怕冷。无论是为了保证设备稳定而常年低温的研究所，还是总飘着漫天大雪的故乡，她只觉得这个温度很舒服。  
而父母离开的那天，是伊薇特度过最冷的一天。

引航者，最近在忙些什么呢？  
  
等反应过来时，才发现已经真的，好久没有见到你了。  
大概是因为，我们共同度过的时光，总会在脑海中一遍遍地浮现……  
就总感觉见面的日子，不会太遥远。  
  
在没有和引航者见面的日子里，我做了很多事情：  
鼓起勇气排队买了限定款的奶茶；  
为数据可视化设备征集了一批新的志愿实验对象；  
给菲做了新的衣服，虽然它看上去有点为难……但还是收下了。  
  
  
此时我才意识到，引航者已经参与了我生活中的这么多部分，  
已经有这么多事情，都成为了我们共同的回忆……  
  
我怀念那些时光，但人是不可能只靠过去的时光活着的。  
我想要和引航者，一起创造更多美好的回忆。  
尽管笔触笨拙，我还是写下了这封信……  
因为我有预感，离见面的日子，真的不会太遥远了。
