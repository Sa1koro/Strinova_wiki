## 角色小传

### 新的宇宙

|  |
| --- |
| **解锁条件：**达到0好感度解锁 |

对于星绘来说，母亲曾经是自己的“引航的星星”，她的期许就是自己成长的动力。而母亲也以身为女儿的榜样为傲。为了让星绘更好的成长，母亲忘我地工作，积攒一切可用的资源。一切都是为了让女儿通向自己幻想中的那个光辉的未来。  
在还是孩子的星绘眼中，母亲聪慧、强大，虽然忙碌但从不疏于指导女儿成长。受益于母亲辛苦积攒下的资源，星绘能比同龄人更早地培养自己的兴趣，接受优质教育，也拥有更多选择的自由。  
当星绘第一次表现出对天体和宇宙的兴趣后，母亲就带着她来到了欧泊设立的大型天文台——这里是母亲工作的场所。  
凝望着女儿摆弄精密仪器，手法笨拙，却似乎又无师自通的模样，母亲说出了星绘此后的人生无法忘却的那句话——  
“星绘，你要去成为那颗在夜空中最为闪耀的星星。”   
此后，在母亲的指引下，星绘入学、读书、跳级进入大学，再以优异的成绩提前毕业，最后如愿以偿地进入了欧泊。  
而就在同年——“崩溃”的前兆，已经悄然发生。

### 裂缝

|  |
| --- |
| **解锁条件：**达到200好感度解锁 |

首先是小规模的、区域化的意识模糊现象被反馈在居住区的常规医疗档案中。作为管理机构的“欧泊”十分重视这些报告，及时指派了救助人员。  
然而这些救助几乎没有取得任何效果。就在医疗界尚在争论不休时，“崩溃”开始大规模爆发了。  
当时，作为“欧泊”的一员，星绘也参与了相关的救助工作，却对“崩溃”束手无策。  
随着“剪刀手”组织的登场，正式引爆了“新世界”积累的所有矛盾。  
对现状开始感到迷茫的星绘求助于母亲——此时的母亲已经就职于欧泊的高层部门——母亲的回答，是让星绘继续钻研最前沿的学术和人脉经营。为关心太多无关紧要的人和事分心，是她最不愿意在女儿身上看到的。  
去成为最闪耀的那颗星星吧，星绘！

### 黯淡

|  |
| --- |
| **解锁条件：**达到500好感度解锁 |

无论当时是基于怎样的想法，是对母亲长久以来精心呵护与培育的反叛，是看到学术研究面对苦难爆发的局限性，以及自己缺乏行动的焦虑，甚至包括自己对这片宇宙茫然无知……  
星绘也无法从自己纷杂的思绪中得出一个准确答案。也许都并存于自己的心中吧。  
总而言之，星绘离开了欧泊，离开了母亲的身边。  
她基于自己先前寻访和救助“崩溃”患者时的实践经验，总结出了一套“意识修复”医疗方案，再用积蓄开办了“星庇所”，就此开始帮助一些寻医无门的患者。  
话说回来，原本一直致力于天文和宇宙学相关研究的自己，为什么会忽然投身于这样的工作？一向头脑清晰的星绘一时也给不出答案……  
偶尔，星绘能感受到内心中有一个声音在质询自己。  
……  
星庇所在勉力维持了一年后，因为资金不足而倒闭了。

### 前行

|  |
| --- |
| **解锁条件：**达到900好感度解锁 |

几位访客为星绘提供了一线希望：乌尔比诺联合财团的新医疗产业部门正在开展最新一轮的投资活动。  
如果想办法将“意识修复”作为一个新医疗项目推荐给乌尔比诺，至少能够解决资金方面的燃眉之急。  
这几位患者曾经表现出轻至中度不等的意识崩溃前兆，幸亏有星绘及时介入，锚定了他们的自我意识，控制了病情。  
如果由他们出面证明“意识修复”医疗方案的疗效，或许能够顺利地获得一笔投资。  
有了访客的推荐，星绘与乌尔比诺投资机构的洽谈过程短暂且顺利。她的项目第一次获得了较为正式的认可。  
值得一提的是，星绘顺便参观了乌尔比诺财团的总部（此后她将多次来到这个地方），参观了关于“向多维宇宙进发”的计划。  
不知为何，星绘感觉不到自己对这个计划的任何兴趣，所以几乎没有参与乌尔比诺设立的相关开发项目。她依旧守护着自己小小的星庇所，过着沉默而忙碌的日子。

### 第一颗星

|  |
| --- |
| **解锁条件：**达到1300好感度解锁 |

——“星绘姐姐，为什么我的爸爸和妈妈忽然不记得我了？”  
一位被送到星庇所的小患者曾经向星绘提出这个问题。而那时星绘是如何回答的？  
孩子无法理解复杂的“意识崩溃机制”，他们所能观察到的现象，就是自己身边的某个人会忽然性情大变，随后开始和亲人发生无休止的争吵、摩擦和冲突。一个曾经热闹的家庭完全解体。  
与孩子不经遮蔽的观察及切身的痛苦相比，任何试图缝合伤口的话语都是无力的。  
所以，那时星绘根本给不出答案。她知道，那些孩子的父母亲还活着，但是身为崩溃症患者，他们几乎已经是另外一个人了。而这些孩子呢？他们的未来又该如何呢？  
所以，星绘能做的就是尽力安抚好小患者们的意识状况。看着入睡后的孩子们的小脸蛋，想象着他们的意识深处的“伤痕”，能通过自己的双手和话语，一点一点地愈合。  
乌尔比诺的投资解决了一些资金问题，但星绘仍然缺乏同伴，只能加大工作强度，加班熬夜成了家常便饭。为此，星绘每天都要调配星芒咖啡——一种苦甜交织，提神醒脑的饮料——慢慢的，这成了她小小的乐趣之一。  
星庇所的名字渐渐地不再鲜为人知，接受过星绘帮助的人亦不会遗忘她的温柔。但星绘依旧形单影只。  
直到“引航者”的出现……  
被患者们视为“星星”的星绘，也将会迎来属于自己的“第一颗星”吧？

等到下一次最佳观测期，前辈能和我一起去看看它吗？  
要是还能和您一起，坐在秋夜下，一颗颗地数着天际的星星……那该多好啊。
