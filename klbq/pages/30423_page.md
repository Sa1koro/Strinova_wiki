## 武器数据

|  |  |
| --- | --- |
| 名称 | 防弹屏障 |
| 种类 | 战术道具 |
| 获得方式 | 账号等级3级解锁 |
| 介绍 | 投掷一个防弹装置，展开能够阻挡子弹的屏障，血量归零时自动销毁。 半径约4.5米的球形护盾。血量为700点，随时间衰减。衰减速率为100点/秒，血量归零后自动摧毁。 护罩会被判定为障碍物，但仅能阻挡双方子弹，无法阻拦双方的技能、道具与行动。 装置本体无法被摧毁。 防弹屏障没有最大滞空时间，理论上可以扔无限远。 |
