## 模式设定

- 支持引航者编辑自己的地图，支持五大玩法模式：爆破入侵、晶源感染、无限团竞、极限刀战、晶源追击。
- 支持多人共同建造，支持下载别人的地图，支持自定义房间规则参数设置。引航者可以由此自己搭建出多种多样的关卡！
- 多达55种模块：包含功能模块、基础模块和装饰模块，快来打造独属于你的地图吧！

### 地图工坊入口

- 点击【娱乐大厅】-【地图工坊】即可进入地图工坊

    :   [![地图工坊教学2.png](https://patchwiki.biligame.com/images/klbq/thumb/3/36/q07ud5r0xo2gcvvwjvs9f1vk8hehetq.png/833px-%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A62.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A62.png)

### 局内建造和保存

- 请注意：你必须摆放齐全这个模式必要的所有模块，才能保证这个地图在保存后能正常提交和试玩；

:   :   [![地图工坊教学5.png](https://patchwiki.biligame.com/images/klbq/thumb/4/41/e5sm79kynuwjqeqccaeejerwhqxxw6y.png/964px-%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A65.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A65.png)

- 局内建造的快捷键说明：“记得使用P按键，给你的地图截取好看的封面吧！”；

:   :   [![地图工坊教学6.png](https://patchwiki.biligame.com/images/klbq/thumb/2/24/t9nibejr280xapdljaonswivsriezqo.png/536px-%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A66.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A66.png)

- 主动保存：当你点击【ESC】时，记得点击【退出并存至草稿箱】按钮，才能将此地图保存到你的【草稿箱】中，请仔细保存你的建造成果哦；

:   :   [![地图工坊教学7.png](https://patchwiki.biligame.com/images/klbq/b/bd/5fb21ykohod96j59dv4vpc950x59hyl.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A67.png)

- 自动保存：每过一段时间，会触发提示，将正在建造的地图自动保存到你的【草稿箱】中。

:   :   [![地图工坊教学8.png](https://patchwiki.biligame.com/images/klbq/e/ed/jaxhg4cmdotvbalf037i98yqrug6v1k.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A68.png)

### 编辑你的历史地图

- 在【草稿箱】中选择你的历史地图，【双击】或点击右下角的【编辑地图】，即可再次打开该地图继续编辑。编辑完的历史地图都会保存在你的【草稿箱】中；

PS:“引航者们可以多次编辑你的地图，用耐心慢慢打造属于你心目中的完美地图哦~”

:   :   [![地图工坊教学9.png](https://patchwiki.biligame.com/images/klbq/thumb/f/fa/1q37mgl0xdyg4k68n1igm2c026hihfx.png/1002px-%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A69.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A69.png)

- 你已经提交的云端地图，仍可以邀请好友们来进行二次编辑，编辑好的地图都会保存在【草稿箱】中，同时需要从【草稿箱】中再次提交此地图，方可发布新的地图。（已提交的地图不会被新的所覆盖哦~）

:   :   [![地图工坊教学10.png](https://patchwiki.biligame.com/images/klbq/thumb/d/d1/rami086d29bez7p04aa08trnzvvp5pl.png/1047px-%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A610.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A610.png)

### 提交地图（提交后会进入审核阶段）

- 点击右键选中菜单栏的【提交】，即可将【草稿箱】中的地图提交到【云端】（提交到云端，才可以跟好友们一起试玩自己的地图）。
- 提交地图进入【审核中】状态后，根据地图复杂度和可能存在的违规内容决定审核的时间，通常在24小时内审核完成并返回结果，请引航者耐心等待。

:   :   [![地图工坊教学11.png](https://patchwiki.biligame.com/images/klbq/thumb/3/3e/11fketzfna9pic0uyfrge82i0kn75jk.png/1021px-%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A611.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A611.png)

### 发布/下架地图

- 将已经在【云端】的地图，发布到【自创地图】的大厅页面，让全服的玩家均能体验你的地图；

:   :   [![地图工坊教学12.png](https://patchwiki.biligame.com/images/klbq/thumb/4/4b/ilxup6b7xrct1nhyhufaulbmjgu3zro.png/1010px-%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A612.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A612.png)

- 设置好你的规则！！（跟原有的自定义房间设置规则的流程一致），输入你发布这张地图的对应玩法模式名称、描述，以及设置好这张图下载权限（设置成允许下载时，所有引航者都可以下载你的地图）；

:   :   [![地图工坊教学13.png](https://patchwiki.biligame.com/images/klbq/thumb/3/34/otccm9c1b6krp76xbygkt4typfn43nn.png/1017px-%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A613.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A613.png)

- 点击【云端地图】-右键选中菜单栏的下架，即可下架你已发布的地图。

:   :   [![地图工坊教学14.png](https://patchwiki.biligame.com/images/klbq/thumb/f/f5/nk82xtx6rmwmrygjkkzdycykynhcuz9.png/1038px-%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A614.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A614.png)

### 体验服【草稿箱】地图保存方法

- 在文件资源管理器中，打开显示隐藏的项目
- 点击【查看】菜单--点击【显示】--勾选【隐藏的项目】

:   :   [![地图工坊教学22.png](https://patchwiki.biligame.com/images/klbq/thumb/4/46/ar71gl2r2pufmmml4brtx939ngo71uz.png/912px-%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A622.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A622.png)

- 打开卡拉彼丘游戏客户端，进入【娱乐大厅】--【地图工坊】
- 点击【草稿箱】界面，右下角点击【导入地图文件】

:   :   [![地图工坊教学23.png](https://patchwiki.biligame.com/images/klbq/thumb/c/c8/gxoh6nmmjz9d7r4rdgydmj6y1dr35jw.png/927px-%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A623.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A623.png)

- 在弹出的路径选择窗中，找到UGCMap文件夹，默认路径为：

C:\Users\【计算机用户名】\AppData\Local\CalabiYau\Saved\UGCMap

- 在打开的文件夹中选择要导入的地图文件（其格式为DATA），点击【打开】进行导入

:   :   [![地图工坊教学24.png](https://patchwiki.biligame.com/images/klbq/thumb/9/9a/tavomf169j4g7zi3ohse1mpquh0ugk5.png/887px-%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A624.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E5%9C%B0%E5%9B%BE%E5%B7%A5%E5%9D%8A%E6%95%99%E5%AD%A624.png)

- 导入完成后，在草稿箱中可看到导入地图，然后就可以继续创作了喵！

新增玩法：晶源追击：

- 玩法简介：
  - 超弦体成功抵达撤离点并坚守一定时间后即可获得胜利
  - 晶源体需要在超弦体撤离前感染所有超弦体，或在超弦体之前率先抵达检查点/撤离点
- 玩法调整：

:   :   [![公告-20251125公告2.png](https://patchwiki.biligame.com/images/klbq/thumb/6/60/blmnc5qw6bsbd7jbcyaq0j5ymxkgssv.png/428px-%E5%85%AC%E5%91%8A-20251125%E5%85%AC%E5%91%8A2.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E5%85%AC%E5%91%8A-20251125%E5%85%AC%E5%91%8A2.png)

- 新增功能：
  - 引航者可以在自定义房间规则面板，设定枪械的击退系数与减速系数
- 新增追击专属功能组件【初始复活点】
  - 所有引航者在回合初始时的复活点。选中区域后，按F在其附近放置其对应复活点。

:   :   :   :   [![公告-20251125公告3.png](https://patchwiki.biligame.com/images/klbq/thumb/0/06/faiaevnjkthpmyc42uaf9axqgor21dc.png/554px-%E5%85%AC%E5%91%8A-20251125%E5%85%AC%E5%91%8A3.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E5%85%AC%E5%91%8A-20251125%E5%85%AC%E5%91%8A3.png)

- 新增追击专属功能组件【检查点&复活点】
  - 超弦体每激活1个检查点，将同时激活其对应的复活点。
  - 当晶源体抵达了检查点，超弦体本回合将失败。
  - 选中检查点后，按F放置其对应复活点
  - 选中检查点后，按G配置检查点的激活顺序。
  - 每个检查点的激活顺序需要连贯接续，不可跳序配置。

:   :   :   :   [![公告-20251125公告4.png](https://patchwiki.biligame.com/images/klbq/thumb/e/ed/1zk4p1toezil3nm0y6rvydn2mcoqey0.png/656px-%E5%85%AC%E5%91%8A-20251125%E5%85%AC%E5%91%8A4.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E5%85%AC%E5%91%8A-20251125%E5%85%AC%E5%91%8A4.png)

- 新增追击专属功能组件【超弦体撤离点】

:   :   [![公告-20251125公告5.png](https://patchwiki.biligame.com/images/klbq/thumb/6/63/ntwkdnnrc4k9pa4gub7jkwpof4yos39.png/638px-%E5%85%AC%E5%91%8A-20251125%E5%85%AC%E5%91%8A5.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E5%85%AC%E5%91%8A-20251125%E5%85%AC%E5%91%8A5.png)

- - 超弦体抵达后将获得胜利，反之若被晶源体抵达则超弦体将失败。
  - 选中撤离点后，按G配置其激活顺序（需放于最后）；
  - 选中撤离点后，按G也可配置其获胜触发延迟时间。
- 追击超弦体补给箱/晶源体晶粹
  - 与晶源感染模式的补给箱和晶粹功能类似，但每回合仅可被拾取1次，不会再次生成。
