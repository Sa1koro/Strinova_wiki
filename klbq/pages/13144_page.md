## 米雪儿

|  |  |
| --- | --- |
|  | 一打开拍摄人员名单，我立刻锁定了老熟人的名字。这次，米雪儿将作为志愿者代表进行宣传照拍摄！ |
|  | 平日里，热情的欧泊搜查官总为我提供帮助，这次，以米雪儿做为矛盾突破口，应该也不会错。 |
|  | 况且，能让米雪儿炸毛并失联的事……实在让人很好奇啊！ |
|  | 这孩子好像一直没有看通讯设备，该去哪里找她呢？ |
|  | 摊开园游会的导览地图，我的目光立刻被一处地点吸引。 |
|  | 嚯，没有比这里更适合米雪儿的地方了！ |

直行，右转，再走个五十米……到了！   
我仰头，巨大的“电玩中心”招牌在阳光下熠熠生辉。   
电玩中心！孩子们的梦想！游戏玩家的天堂！米雪儿的快乐老家！以我对她的了解，她不可能不在这里。   
  
推开虚掩的门走进去，一眼就看到米雪儿盘腿坐在游戏机前，屏幕上的战斗已进入白热化阶段。   
“伙伴！你怎么来了？不不不，你来真是太好了！”余光瞥到我，米雪儿疑惑的眼神一秒转为惊喜，疯狂努嘴示意桌上还有一个手柄：“快来帮忙，我一个人打不过这关！”   
好吧，虽然我是带着任务来的，但先浅打两把，应该没什么影响吧！   
  
待我和米雪儿满足地放下手柄时，一个小时已经过去了。  
“伙伴——你简直就是天降神兵啊！我已经卡在这里很久了！”米雪儿抱着通关奖励——火力大喵玩偶，激动地在原地转了个圈：“不过你怎么会出现在这里呀？我以为现在只有工作人员呢。”   
我忙把摄影师交代给我的任务转述给米雪儿。   
“矛盾？我不记得大家有什么矛盾啊，不过关于宣传方向，群里确实还没讨论出结果。闲着也是闲着，我就偷溜出来打游戏了。”米雪儿不好意思地挠头：“需要我的时候，我会随叫随到的，这不算摸鱼吧。”   
“宣传方向……”我琢磨着她话里的字眼，大概有了点想法：“那米雪儿觉得该怎么宣传呢？”   
“我来宣传的话，一定会狠狠夸这个空间超大、设备超多的电玩中心啦！哎呀，我当然知道不合适，所以就不参与讨论了。”米雪儿眨眨眼，“难题就交给她们去解决吧，伙伴，我们再开一把？”   
听上去似乎是一场误会……但事实果真如此吗？   
无论如何，Mission One，Completed！

## 梅瑞狄斯

|  |  |
| --- | --- |
|  | 庆典游园会的总设计师梅瑞狄斯女士，在业界声名赫赫，她所策划的展会都以实用主义的功能性和简洁的美学设计而闻名。 |
|  | 印象中，这还是她第一次接手庆典活动的设计。不知道理性与感性会在此碰撞出怎样的火花呢？ |
|  | 我已经开始充满期待了。 |
|  | 梅瑞狄斯女士，你好，我是负责宣传照拍摄的工作人员，现在方便和我见一面吗？有点事想和你确认一下。 |

| [夏日音乐节2024头像 133.png](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E5%A4%8F%E6%97%A5%E9%9F%B3%E4%B9%90%E8%8A%822024%E5%A4%B4%E5%83%8F_133.png) | 梅瑞狄斯：我现在不在办公室，你来A-2游乐区找我吧。 |

梅瑞狄斯给出的地址，是整个游乐园海拔最高的地方——山顶摩天轮。   
想要到达那里，我必须穿越整个场地。一路走来，能看到许多挂着“已完工”标牌的游乐设施，主步道两边也摆满了空摊位，庆典开始后，这里应该会热闹非凡吧。   
乘坐观光电梯来到约定地点，梅瑞狄斯倚靠在栏杆上，沉静地望着下方。我走过去，顺着她的视线往下看，试图通过共享视野的办法，来猜测设计师在想什么。   
“此刻，它是静止的齿轮组，只等几天之后，涌入的人群作为润滑，让它运转起来。但太过拥挤也会导致阻塞，你知道的，我一向不喜欢人多的地方。”   
“就像你被委托拍摄的宣传照，我——我不太明白，无论是物件造型、结构排布，还是顶灯打在水面上反射出的波纹，都已经足够完美了。还要我们做什么呢？”梅瑞狄斯转向我，手托下巴，似乎真的很疑惑。   
萧条冷清的庆典，空无一人的泳池，没有观众的舞台——这副场景，怎么想都很诡异吧！这就是天才的独特审美吗？   
我看向梅瑞狄斯的眼神不禁带上了一丝敬佩。   
  
“但那样会很孤独吧？”我思索良久：“过山车上不能握住身边人的手，泳池里不能跟着节奏一起摇摆……宣传照的布景再完美，无法展现人们身在其中的快乐的话，也不会有吸引力的。”   
“梅瑞狄斯邀请我来这里，也是想让我看看完整的设计吧？你一定为此感到骄傲。”   
梅瑞狄斯脸上划过一丝惊讶，似乎她从未真的想过这个问题。   
“你说对了，引航者。我确实为每一次优秀的设计感到骄傲。”半晌，她向我扬起头：“作为让我意识到这件事的回报，我会配合你拍宣传照的。”   
“但现在，陪我一起欣赏静止的这里吧。”

## 加拉蒂亚

|  |  |
| --- | --- |
|  | 加拉蒂亚，来自乌尔比诺旗下音乐厂牌的知名DJ，也是本次庆典的重磅嘉宾之一。 |
|  | 内部宣传手册里提到，在泳池派对上，加拉蒂亚也会换上泳装，甚至会走下DJ台，和现场观众互动。 |
|  | 除了DJ外，这位女士还有诸多身份。光我知道的就有魔术师、艺术品商人等等……真的有人能同时经营这么多身份吗？该不会她其实有分身术吧…… |

|  | 引航者：加、加拉蒂亚小姐？您什么时候来的？吓我一跳…… |

|  | 引航者：（真的是惊喜不是惊吓吗……） |

即使什么也不做，只是站着，面带笑容的加拉蒂亚就让人无法移开视线。   
热情而浪漫，潇洒而自由，这样的加拉蒂亚，即使拥有再多身份也不会让人觉得奇怪——她看上去就不像会被某一个定义束缚的人，我在心里将之前对她的揣测默默画了个叉。  
所以，是什么能让她陷入一个关于宣传照拍摄的矛盾呢？总不会是被人抢了C位吧。要这么说就太小瞧加拉蒂亚了，她是站在最后一排也能大放光彩的人。   
  
“怎么找到了我又一言不发呢？”加拉蒂亚向我伸出手，“不知道怎么开口的话，就一起散散步吧。”   
林荫道上树木繁茂，遮挡了大部分阳光。加拉蒂亚踩着地上的斑驳光影，看上去非常愉快。   
“您似乎心情很好呢。”我忍不住开口，“所以关于宣传照拍摄的争执，只是个误会对吧。”   
“不是误会哦。”加拉蒂亚笑眯眯地回答。   
“嗯……啊？”我愣住了。   
“官方想要把泳池派对作为最大的宣传点，为我设计专门的海报。但这样的话，就太无聊了。”宽大的遮阳帽被取下，在加拉蒂亚的指尖翻飞：“比起现场挤满慕名而来的观众，我更希望陌生人在派对后变成加拉蒂亚的追随者。”   
“这才有趣嘛。”   
面前突然出现一朵橙色的小花，是……加拉蒂亚从帽子里变出来的？   
“送给你。我会换上全新的装扮，所以，在宣传照上隐去名字这件事，就拜托你了。”

## 芙拉薇娅

|  |  |
| --- | --- |
|  | 和其他三人沟通完后，我已经不知道这究竟算矛盾还是算乌龙了。 |
|  | 但可以确定，这是个除了“AAA专业宣传照拍摄”外没有人受伤的世界…… |
|  | 啊，还要去找一下芙拉薇娅……可是一路上都没有看到人啊。 |
|  | 通讯设备里躺着芙拉薇娅传来的讯息，说在“全世界最快乐的地方”等我。 |
|  | 几乎每个设施都走过一遍了，再走就要回大门口了，还是没有看到芙拉薇娅的影子。 |
|  | 嗯？那不是……？！ |

与平常不太一样的芙拉薇娅站在门口，紫色长发扎成高马尾，身穿一套条纹棒球服，看上去像是准备开球的运动系学姐：“欢迎来到全世界最快乐的地方，请出示门票。”  
呃，这是什么新型的Cosplay吗……我不知所措地举起了摄影小哥给我的工作证。  
“门票查验通过，我是你的游园向导蝴蝶，请跟紧我，我来为您介绍整个庆典场地。”   
整个场地？我才刚走过一遍啊！   
“那、那就麻烦蝴蝶小姐了！”既来之则安之，我咬着牙答应了下来。   
  
跟随着芙拉薇娅，我再次走过了游乐区、商业街、休闲区，虽然不间断的行走让人疲惫，但芙拉薇娅细致的讲解还是很有趣的，梅瑞狄斯的设计隐藏着如此多的巧思啊……  
最后，我们来到了泳池边，此时，我已经疲惫不堪了。芙拉薇娅仿佛读出了我心中所想，手放在我的肩上轻轻使力，我立刻顺势瘫倒在躺椅上，发出了一声喟叹。   
——好软的椅子！   
芙拉薇娅的声音从头顶传来：“这一路感觉怎么样？”   
“走了很久，太阳很晒……和你比赛打气球、捞金鱼都输了……但是整体很愉快。现在躺着，也很舒服……”   
“哦？可你看起来很累呢。”   
“累，并快乐着……”   
“这就是庆典的珍贵之处哦。入园时期待的心情，赢和输带来的喜悦和失落，走在路上觉得吵闹也好，和心上人一起忽略了其他人也好，结束行程后的疲惫也好，‘游园会’本身就应该是乐趣所在。”悦耳的声音继续诉说：“同理，如果要拍摄宣传照的话，没有比身处其中、并享受这里的游客更适合的对象了。”   
懂了，我完全懂了！我抬起手，对芙拉薇娅比了个懒洋洋的OK，换来后者一声轻笑。   
“享受拍摄宣传照前最后的休息时间吧，我和你一起。”
