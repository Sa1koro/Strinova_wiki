## 武器数据

|  |  |
| --- | --- |
| 名称 | 警报器 |
| 种类 | 战术道具 |
| 获得方式 | 账号等级18级解锁 |
| 介绍 | 投出后附着在障碍物表面约1.5秒后生效，在一段时间内感应其周围，有敌人经过时发出警报并对其造成10%减速效果。 感应半径为7米，感应持续时间约为120秒，可隔墙感应。 有敌人时警报器上方会出现感叹号，并伴随着滴滴声；敌人进出警报器时会获得内容为“X人触发报警”的屏幕居中文字提醒（X为警报器范围内的敌人数量）。 护甲值为10点，本体可被摧毁。 警报器没有最大滞空时间，理论上可以扔无限远。 |
