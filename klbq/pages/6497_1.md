### 背景故事

|  |
| --- |
| **第一赛季——** |
| **赤刃行动** |

[![壁纸-第1赛季赤刃行动.png](https://patchwiki.biligame.com/images/klbq/thumb/f/fd/oj8u6x8hkl9b9dji632vkyen9p7tuvo.png/700px-%E5%A3%81%E7%BA%B8-%E7%AC%AC1%E8%B5%9B%E5%AD%A3%E8%B5%A4%E5%88%83%E8%A1%8C%E5%8A%A8.png)](https://wiki.biligame.com/klbq/%E6%96%87%E4%BB%B6:%E5%A3%81%E7%BA%B8-%E7%AC%AC1%E8%B5%9B%E5%AD%A3%E8%B5%A4%E5%88%83%E8%A1%8C%E5%8A%A8.png)

  

纵然是如此理想乡，卡丘依旧存在着名为崩溃症的一种隐患，它会影响所有进入这个多维世界的生物基因，人类会产生诸如躁狂、精神失常等症状，而其他生物则会逐渐衰弱而死。  
最终欧泊推出了通过“轮回计划”这一项目，用特殊的能量场配合频段信号遮蔽了卡丘世界的混乱能量，但也造成了二次伤害，人类会像是硬盘一样不断格式化记忆。  
  
得知这一秘密的一部分卡丘人深感不安与愤怒，因此高举反旗选择对抗，用意保留珍贵的记忆，同时有人打出了重返地球这一“失乐园”的旗号，怀揣着各种信念的人被拧成了一股麻绳，成立了“剪刀手”。  
  
脱胎于欧泊的财团组织“乌尔比诺”则是在双方的对抗中暗中积蓄力量，希望将卡丘作为跳板，以自由、科学、艺术打造新技术，走向新世界。  
  
三方为了各自目的，开发和争抢新世界的资源而再度抗衡。
