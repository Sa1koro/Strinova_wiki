|  |  |
| --- | --- |
| 名称 | 风场雷 |
| 种类 | 战术道具 |
| 获得方式 | 账号等级20级解锁 |
| 介绍 | 落地1秒后激活，制造一小片足以将人吹起的风场，使风场中的所有单位强制飘飞。 持续时间为5秒。影响范围为半径5米，高度6米的圆柱体，最高可吹起高度为7米。 风场雷没有最大滞空时间，理论上可以扔无限远。 |
