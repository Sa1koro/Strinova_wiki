|  |  |

| 名称 | 治疗雷 |
| 种类 | 战术道具 |
| 获得方式 | 自动获得 |
| 介绍 | 在投出5s或受到撞击后激活，展开一片持续时间约3秒的治疗云雾，令其范围内的我方单位获得生命恢复Buff，效果为恢复生命值。。 Buff持续时间为15秒，恢复速率为4点/秒。 受伤后的2秒内处于战斗状态，此时治疗效果不会生效，且会正常消耗持续时间。游戏中的大部分持续恢复类效果均如此，不会被受伤打断生命值/护甲值恢复的效果均有特殊说明。 在治疗云雾内不会持续刷新治疗效果的持续时间，获得治疗Buff后离开即可。 |
